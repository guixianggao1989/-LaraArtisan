# -LaraArtisan

LaraArtisan 是一个功能强大的 Laravel Artisan 命令增强工具包，旨在扩展和简化 Laravel 命令行工具的使用体验。它提供了一系列实用的自定义命令、开发辅助工具和生产力提升功能。
✨ 核心特性
🚀 增强的开发命令
智能代码生成
支持 Model / Controller / Request / Resource / Policy批量生成
内置多套可切换模板（Admin / API / CRUD / DDD）
自动补全命名空间、路由、表单验证规则
数据库工具
一键生成测试数据（支持 Faker 本地化）
表结构差异对比（migrate:diff）
字段级迁移生成器
系统监控
artisan app:status（环境、缓存、队列、连接）
请求耗时 / 内存占用 / 慢查询统计
批量操作
批量替换命名空间、类名、表前缀
批量清理未使用模型、控制器、路由
🔧 开发效率工具
项目脚手架
快速初始化 Admin / API / Microservice 架构
可选集成 Laravel Octane / Sanctum / Telescope
代码优化
PHPStan + Rector 一键质量扫描
Artisan 自动格式化（artisan code:fix）
依赖管理
Composer 安全检测 & 过期包提醒
国内镜像切换助手
部署辅助
服务器环境检查清单
一键维护模式 + 平滑发布
Env 配置差异对比
